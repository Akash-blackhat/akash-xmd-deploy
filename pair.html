How to get Session ID:
  • 1️⃣ Click "Open Pairing Portal" button above
  • 2️⃣ Enter your WhatsApp number with country code
  • 3️⃣ Copy the pairing code or scan QR code
  • 4️⃣ Open WhatsApp → Settings → Linked Devices
  • 5️⃣ Tap "Link with phone number instead"
  • 6️⃣ Enter the pairing code
  • 7️⃣ Your Session ID will be sent to your WhatsApp
  • 8️⃣ Copy the DKML~ session ID and use in Deploy page
